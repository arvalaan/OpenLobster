# Browsing the Knowledge Graph

The left sidebar of the Memory view is your primary navigation tool. It organises all memory nodes by their type and lets you search across the entire graph.

## How nodes are organised

Memory nodes are grouped by their `type` field (for example: `Person`, `Organisation`, `Topic`, `Event`). Types are discovered dynamically from the graph data and displayed in alphabetical order.

To navigate:

1. Expand a type group by clicking its header to reveal the nodes it contains.
2. Click a node name to open its full details in the right panel.

## Searching for a node

The search box at the top of the sidebar filters nodes in real time as you type. You can search by:

* Node label (name)
* Node type
* Property content

{% hint style="info" %}
If many nodes match your query, include part of the type name to narrow results — for example, searching `project` will surface nodes of type `Project` before less relevant matches.
{% endhint %}

## After selecting a node

Once you click a node, its details load in the right panel. From there you can read its properties and connections, edit its data, or delete it. See [Node Detail](node-detail.md) for more.
