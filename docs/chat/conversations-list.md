# Conversations List

The left panel of the Chat view lists every conversation the agent has had, from the most recent to the oldest. Each row represents one user session.

## What each row shows

* **Avatar** — For direct conversations, a single circle showing the first letter of the user's name. For group conversations, two overlapping circles to distinguish them visually at a glance.
* **User name** — The display name assigned during pairing.
* **Channel badge** — The platform the user is writing from: Telegram, Discord, WhatsApp, Slack, SMS, or Loopback.
* **Last message time** — When the most recent message in this conversation was sent.

## Selecting a conversation

Click any row to load the full message thread on the right. The selected row is highlighted so you always know which conversation is open.

To switch conversations, click a different row. There is no need to close the current one first.

## Group conversations

When the agent is a member of a group chat (for example, a Telegram group or a Discord channel), those conversations appear in this list alongside direct messages. The channel badge identifies the platform, and the user name reflects the group or channel name.

{% hint style="info" %}
If the list is empty, no conversations have taken place yet. Users must initiate contact through one of the configured channels to create a conversation.
{% endhint %}

## Searching and filtering

The list updates in real time as new messages arrive. If you have a large number of conversations, scroll through the list or look for the active/idle badge in the [Dashboard > Recent Conversations](../dashboard/recent-conversations.md) panel to find the one you need quickly.
